=head1 NAME

cwb-atoi - Generate uncompressed binary CWB data file from ASCII numbers

=head1 SYNOPSIS

B<cwb-atoi> [-nl] I<file>

=head1 DESCRIPTION

B<TODO>

Reads one integer per line from ASCII file I<file> or from standard input and writes values to standard output as 32bit integers in network format (the format used by uncompressed CWB binary data files).


=head1 OPTIONS

  -n  convert to network format [default]
  -l  convert to little endian format
