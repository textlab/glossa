=head1 NAME

cwb-compress-rdx - Compress (token-level) indexes of CWB corpus

=head1 SYNOPSIS

B<cwb-compress-rdx> [-d] [-D I<file>] [-T] [-r I<registry_dir>] [-f <prefix>]
    ( -P I<attribute> | -A ) I<corpus>

=head1 DESCRIPTION

B<TODO>

Compress the index of a positional attribute. Creates C<.crc> and C<.crx> files which replace the corresponding C<.corpus.rev> and C<.corpus.rdx> files. After running this tool successfully, the latter files can be deleted.

B<NB:> The recommended front-end for indexing and compression is the B<cwb-make> program supplied as part of the CWB/Perl interface.


=head1 OPTIONS

  -P <att>  compress attribute <att> [default: word]
  -A        compress all positional attributes
  -r <dir>  set registry directory
  -f <file> set output file prefix (creates <file>.crc and <file>.crx)
  -d        debug mode (print messages on stderr)
  -D <file> debug mode (write messages to <file>)
  -T        skip validation pass ('I trust you')
  -h        this help page
